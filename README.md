# LLMFabric UI

UI for [LLMFabric](https://github.com/6block/llmfabric).

## Installation

1. [Nodejs](https://nodejs.org/en) 16.0+(with NPM)

If you're on Mac

```
brew install node
```

2. [pnpm](https://pnpm.io/installation#using-npm)

```
npm install -g pnpm
```

3. Setup

```
git clone https://github.com/6block/llmfabric-ui/
```

4. Install dependencies

```
cd llmfabric-ui
pnpm install
```

## Usage

1. Run development server (at http://localhost:9000)

```
npm run dev
```

2. build release

```
npm run build
```
